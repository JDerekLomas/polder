## The strategic split (read first)

Open-source funders (NLnet, Sovereign Tech Fund) fund **open digital base technology**,
not proprietary products. So frame applications around the **open-source EU
provider/engine work on `opencode`** (the sovereign inference adapter, audit logging,
no-telemetry fork) — *not* the MakeMode desktop product. The product wrapper is funded by
customers + equity; the **sovereign infrastructure layer is grant-fundable.** Two
different fundable artifacts.

## Recommended sequence

1. **WBSO** the moment the BV exists (cheapest money, funds Emma's hours).
2. **Sovereign Tech Fund** proposal on the open-source engine — apply now, rolling.
3. **Prep the NLnet proposal** so it's ready when the call reopens after summer.
4. **Emma maps TU Delft / Delft AI + AiNed** instruments — the warm, insider channels.
5. Tier 2/3 only once there's pilot evidence + a partner network to apply with.

The pilot evidence (published links, activation, quotes — see `PLAYBOOK.md` §4) is the
single asset that strengthens *every* application above. Run the pilot, then apply from
strength.
